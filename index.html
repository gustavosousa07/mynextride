Sobre o Projeto MyNextRide

O MyNextRide é uma solução de Business Intelligence desenvolvida para o Desafio ClickBus 2025.

Utilizando um conjunto de dados de viagens, a plataforma aplica técnicas avançadas de análise de dados e machine learning para decodificar o comportamento dos clientes, incluindo segmentação, análise de hubs e modelagem preditiva para prever a próxima compra e rota de um cliente.

O objetivo é transformar dados brutos em inteligência acionável, permitindo que a ClickBus otimize suas estratégias de marketing, personalize ofertas e melhore a retenção de clientes.
